# IPL 2022 Data Analysis using Python

## Project Overview

This project presents an exploratory data analysis (EDA) of the IPL 2022 dataset obtained from Kaggle. The analysis was performed using Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn to extract meaningful insights and visualize trends in the tournament.

The project answers more than 20 analytical questions related to team performance, player statistics, match outcomes, and overall tournament trends through data visualization and statistical analysis.

---

## Objectives

* Perform data cleaning and preprocessing on the IPL 2022 dataset.
* Explore team and player performances.
* Generate visual insights using charts and graphs.
* Answer analytical questions based on IPL 2022 data.

## Analysis Performed

Some of the questions analyzed in this project include:

* Which team won the most matches?
* Which player scored the highest runs?
* Which bowler took the most wickets?
* What was the distribution of match results?
* Which venue hosted the maximum matches?
* Team-wise performance comparison.
* Toss decision analysis.
* Run distribution across matches.
* Player performance analysis.

More than 20 analytical questions have been explored using data visualization techniques.
